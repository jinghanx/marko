Marko

An app that doesn't know what it is.

Keyboard-driven. Vaguely overstuffed. On purpose.

  • WYSIWYG markdown — yes, even the tables
  • Finder-style folders that fit in a tab
  • Code editor with vim and emacs keymaps (because both camps asked)
  • Browser and terminal in the same window as this

Launcher

Spotlight, but for our app.

Yes, Raycast exists. Yes, Alfred exists. Built one anyway — wakes with ⌘⌥Space, opens any app in /Applications, runs Marko commands, computes math, falls back to web search. Took a weekend. Felt great.

  • Type a command name to wake the right Marko tab — chat, git, find, http, terminal…
  • Type an app name to open it. Chrome, Slack, Figma. The dock, but smaller and with vibes.
  • Type math, hit ↵, the result is on your clipboard. Calculator.app stays minimized.
  • A hand-rendered text caret that glides between positions, plus a custom pointer dot. Completely unnecessary. We love it.

Git

A Git client. Not Tower. Not Fork. Right there.

We rebuilt SourceTree. In our app. Hunk-level and line-level stage, unstage, discard. Branches, stashes, tags. Commit history with cherry-pick. Fetch, pull, push. The world did not need this. We made it anyway. It's actually pretty good.

  • Click a single line in a hunk to stage just that line. Yes, just that line.
  • "Delete every branch except current and main" in one click. The healthiest button you'll press today.
  • Cherry-pick from history view. Merge / rebase from branches. No terminal incantations required.

Terminal

A real terminal. Now also haunted by an LLM.

xterm.js + node-pty + your actual $SHELL, with all your aliases and prompts intact. Press ⌘I, describe what you want, watch the AI type into your prompt. You hit Enter. Nothing runs without you. We're not monsters.

  • Click a file path in output to open it in a side pane at the right line. Magic.
  • ⌘F to search scrollback · ⌘K to clear · ⌘-click URLs. Standard issue.
  • The ANSI palette is derived from your theme, so the terminal stops looking like it's from a different app

Find in files

It's just ripgrep. With a UI.

⌘⇧F, then a query, glob, and the usual regex/word/case toggles. Results stream in grouped by file. Click a match to open the file at that line. Calling it "novel" would be a stretch. Calling it "fast" would not.

Layout

Splits nested as deep as you'll regret.

Split horizontally (⌘\), split vertically (⌘=), recurse until the panes are postage stamps. Tmux-style workspaces give each project its own pane tree and root folder. Everything survives restart — workspaces, panes, tabs, the URL the browser tab was just on. We probably went too far. You'll like it.

  • Drag tabs to reorder. Drag them between panes. Drag them into the chat as context. Just drag everything.
  • "Now Playing" pill in the titlebar finds audio/video from any tab in any workspace, so you can pause it
  • Same file open in two panes? Edits appear in both. We didn't have to. We did anyway.
