Features
Stack

Eighteen tab kinds. One window.

Every dev surface that usually lives in its own app, addressable by a single keystroke.

✎

WYSIWYG markdown

Milkdown Crepe — slash commands, drag handles, GFM tables, math, code blocks. Toggle rendered / raw / split.

⟨⟩

Code editor

CodeMirror 6 with 30+ language packs, vim mode, theme-aware syntax colours, and live cross-pane edit sync.

📁

Finder-style folders

Drag-select, sort by date / kind / size, section headers, cut / copy / paste, drag-to-move, Quick Look, MX-mouse back/forward.

▍

Real terminal — with AI

xterm.js + node-pty + your $SHELL. ⌘I describes a command in English; the AI types it into your prompt for review. Click any path in output → opens in the editor.

⎇

Full Git client

Status with hunk- and line-level stage / unstage / discard. Branches, stashes, tags, commit history with cherry-pick, fetch / pull / push. Source-tree-equivalent without leaving the window.

✨

Streaming AI chat

OpenAI / OpenRouter / Anthropic / Ollama / LM Studio — any OpenAI-compatible endpoint. Attach files for context (drag, dialog, or open-tab picker). Markdown rendering with copy-code buttons. History sidebar. Keys encrypted via Keychain.

🔎

Find-in-files

Ripgrep streaming. Case / whole-word / regex toggles. Glob filter. Click any match to jump to that line in the editor.

→

HTTP client

Postman-lite request builder. Method, URL, headers, body. Response with status pill, time, size, JSON pretty-print, headers viewer.

✏︎

Whiteboard

Embedded Excalidraw. Canvas matches your active theme. .excalidraw files round-trip. Cross-pane sync.

◯

Embedded browser

Real Chromium webview with back / forward / reload / address bar. The Now-Playing pill in the titlebar lights up when YouTube or Spotify plays.

▤

Format viewers

PDF (Chromium native). CSV / TSV with sortable tables. JSON tree inspector. Side-by-side file diff. Image fit / 100 %. mp3/mp4 streaming with seek.

▰▰

Splits + sessions

Recursive split panes (⌘\ / ⌘=, nest as deep as you want). Tmux-style sessions, each with its own pane tree and workspace root. Drag tabs to reorder.

⌘

Keyboard-first

⌘P fuzzy file search with Recents. ⌘T multi-suggestion path / URL / command palette. ⌘⇧F find-in-files. ⌘⇧/ for the full cheatsheet.

💾

Stays where you left it

Sessions, panes, open tabs, scratch buffers, AI chats, Git/HTTP requests — all survive restart. Now-Playing pill keeps audio alive when you switch sessions.

🎨

15 color themes

Tokyo Night, Catppuccin, Dracula, Gruvbox, One Dark, Nord, Solarized, Rosé Pine, and more. Token colours and the terminal palette derive from each theme's ANSI palette so everything stays coherent.
