Terminal

A terminal that types commands for you.

xterm.js + node-pty + your real $SHELL. Press ⌘I and describe what you want — the AI types the command into your prompt for review. Hit Enter to run. No surprise execution.

  • Click any file path in output to open it in the editor (with line jump)
  • ⌘F search scrollback · ⌘K clear · ⌘-click URLs
  • Themes match the app's color scheme — ANSI palette derived from each theme

Layout

Splits to the depth you want. Sessions for parallel projects.

Split horizontally (⌘\) or vertically (⌘=), nest as deep as you want. Tmux-style sessions give each project its own pane tree and workspace root. Everything — sessions, panes, open tabs, scratch buffers — survives restart.

  • Drag tabs to reorder, drag-drop into the chat as context
  • "Now Playing" pill in the titlebar surfaces audio/video from any tab in any session
  • Cross-pane edit sync: same file open twice, edits appear live in both
