Features
Stack

Built around how you actually work.

Five distinct tools, one consistent surface. Move between them with muscle memory.

✎

WYSIWYG markdown

Live in-place rendering, slash commands, drag handles, code blocks, math, GFM tables. Saves real .md files.

📁

Finder-style folders

Drag to multi-select. Sort by date, kind, or size with section headers. Quick Look on Space. Cut, copy, paste.

⟨⟩

Code editor

CodeMirror 6 with auto-detected language for 30+ syntaxes. Optional vim mode. Theme-aware syntax colors.

▍

Real terminal

xterm.js + node-pty running your real $SHELL. Themes match the app's color scheme. Multiple panes side by side.

◯

Embedded browser

Type a URL in ⌘T → opens as a tab with back/forward, address bar, and a real Chromium webview.

▰▰

Recursive split panes

Split any pane horizontally or vertically. Nest as deeply as you want. Each pane has its own tabs.

🎨

15 color themes

Tokyo Night, Catppuccin, Dracula, Gruvbox, One Dark, Nord, Solarized, Rosé Pine and more. Light + dark variants.

⌘

Keyboard-first

⌘P fuzzy file search, ⌘T autocompleting path or URL, ⌘\ split, ⌘Y htop-style process viewer, ⌘L address bar.

🗂

Workspaces

Bookmark folders you switch between. Auto-restore the last workspace on launch. Reveal active file in tree.
